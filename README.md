## Installation

---

1. Clone the repository:

```
git clone https://github.com/junhyeok9/youth_policy.git
```


2. 클론한 폴더로 이동:

```
cd youth_policy
```

3. 필요한 패키지 설치:
```
pip install -r requirements.txt
```
